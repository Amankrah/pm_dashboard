# Brand assets

Drop the official artwork here using these exact filenames so the UI picks them up automatically. Until you replace them, the files in this folder are neutral text wordmark placeholders.

| File | Used on | Recommended format | Notes |
|---|---|---|---|
| `mcgill.svg` | Form header, login page (light backgrounds) | SVG | McGill primary mark, dark version |
| `mcgill-white.svg` | Dashboard sidebar (dark background) | SVG | McGill mark, white/reverse for dark navy |
| `mastercard-foundation.svg` | Form header, login page | SVG | Mastercard Foundation mark, dark version |
| `mastercard-foundation-white.svg` | (reserved) | SVG | Reverse version if we ever co-brand on dark |

## Sizing

The UI renders each logo at a fixed pixel height (32px sidebar, 40px login, 48px form header). Whatever the SVG's intrinsic viewBox is, it will be scaled to that height — so don't worry about exact dimensions, just make sure the artwork has reasonable padding inside the viewBox.

## Replacing

Just overwrite the file with the same filename and refresh — no code changes needed.
